# 🎥 CAM-ON Tool

> 🚀 A Powerful Camera Access Tool built using Node.js
> ⚡ Created by **CyberWithSuraj**

---

## 🔥 Features

✨ Real-time camera interaction
✨ Fast & lightweight Node.js backend
✨ Auto dependency installer (CAM-ON.sh)
✨ Clean frontend (HTML + JavaScript)
✨ Beginner-friendly & easy to use

---

## 📦 Installation Guide

### 🔧 Step 1: Clone Repository

```bash
git clone https://github.com/CyberWithSuraj/CAM-ON.git
cd CAM-ON
npm install
```

---

### ⚙️ Step 2: Run Tool (Recommended)

```bash
chmod +x CAM-ON.sh
./CAM-ON.sh
```

---

## 🚀 Usage

✔ Open browser
✔ Allow camera permission 📸
✔ Start using the tool
